# peer-learning-and-tutoring-platform-AF-project
Peer-to-peer learning platform for school students (grades 6–13). Find peer tutors by subject/grade, book video sessions, use interactive whiteboard + chat, share notes, rate sessions, and learn together safely. Mobile-friendly web app made with MERN stack: MongoDB, Express.js, React.js, Node.js. Helping students help each other!
